random-a11y-combo

Get a random accessible color combination

v1.0.0

Installation

npm install --save random-a11y-combo

Usage

const randomA11yCombo = require('random-a11y-combo')

randomA11yCombo()  // => ['#4e9d4d','#0b2352']
randomA11yCombo('#fff')  // => ['#fff','#444']
